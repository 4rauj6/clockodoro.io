# pomodoro-study-appweb
<p>The Pomodoro's study method it's one of the most famous study methods that exists nowadays. In it's core formula, the Pomodoro is a time-management tecniche that has as it's main goal, optimize your studies sections in three different ways: Pomodoro (standard), Short Break (5-10 minutes) and Long Break (20-25 minutes).</p>
